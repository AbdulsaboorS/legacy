  <action>Create capacitor.config.ts at the project root with this exact content:

```typescript
import type { CapacitorConfig } from '@capacitor/cli';

const config: CapacitorConfig = {
  appId: 'app.joinlegacy',
  appName: 'Legacy',
  webDir: 'public',
  server: {
    url: 'https://legacy-bice.vercel.app',
    cleartext: false,
  },
  plugins: {
    StatusBar: {
      style: 'DEFAULT',
      overlaysWebView: false,
    },
    Keyboard: {
      resize: 'none',
      style: 'DEFAULT',
    },
  },
};

export default config;
```

Key decisions:
- webDir: 'public' — the public/ directory already exists; Capacitor CLI requires this directory to exist even when server.url overrides it at runtime. Do NOT use 'out'.
- cleartext: false — Vercel serves HTTPS; cleartext is only for local dev.
- Keyboard resize: 'none' — prevents the WebView from being pushed up when the keyboard appears on refine/onboarding screens. Do NOT use 'body'.
- StatusBar overlaysWebView: false — status bar sits above the app, not over it.</action>
